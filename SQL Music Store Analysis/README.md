# SQL Project Music Store Analysis
SQL project to analyze online music store data

## Objective:
The dataset consisted of information about employees, invoices, customers, tracks, genres, and countries. The goal was to extract valuable insights and provide answers to various ad-hoc questions related to the store's operations and customer behavior.

## Datasource :
Dataset used for this [Music_Data](https://github.com/kirannavale/Portfolio-Projects/tree/main/SQL%20Music%20Store%20Analysis/Data)
